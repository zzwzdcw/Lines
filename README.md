# Lines

## 本软件是一款台词图生成工具

## 使用说明

1.点击添加头图，添加一个想要完全展示的截图

2.点击设置台词范围，将会设置后续台词图的范围

3.点击添加台词图并保存，点击相册，选择上述两个步骤中未选择的台词图，APP将会自动裁剪并保存到Lines中
